# Custome My T-Shirt

## Description
You can costimize your T-Shirt with your own design. You can choose the color, add image and texture.

## Stacks

### Frontend
- React
- Tailwind CSS
- Three.js

### Backend
- Node.js
- Express
- OpenAI

## How to run

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm start
```
